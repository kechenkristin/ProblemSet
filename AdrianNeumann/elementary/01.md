Write a program that prints ‘Hello World’ to the screen.

- java
```java
class HelloWorld() {
	public static void main(String[] args) {
		System.out.print("Hello World");
	}
}
```

- python
```python
print("Hello world")
```

- javascript
```javascript
console.log("Hello World")
```

- go
```go
package main
import "fmt"

func main() {
	fmt.Println("Hello World")
}
```
